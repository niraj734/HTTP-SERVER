# Simple Node.js HTTP Server

This is a basic backend project I made using Node.js core `http` module.  
It creates a simple server that listens on a port and sends a response when someone hits it.

---

## What this project does

- Creates a server using Node.js
- Handles incoming requests
- Logs every request in the terminal
- Sends a simple response back to the browser

---

## How to run it

### 1. Install Node.js (if not installed)

### 2. Run the file

```bash
node server.js
